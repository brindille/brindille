No framework boilerplate
========================

Boilerplate for single page website without using framework.

Lifecycle
=========
1. `bind()`
2. `addEvents()`
3. `createPromises()`
4. `ready()`
5. `onPreloadComplete()`
6. `transitionIn()`
7. `unbind()`
8. `transitionOut()`
9. `beforeDestroy()`
10. `removeEvents()`
11. `destroy()`
